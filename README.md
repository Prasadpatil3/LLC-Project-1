# 📝 Smart To-Do List Pro (Python)

A simple and efficient To-Do List application built using Python.  
This project helps users manage daily tasks, track completion status, and improve productivity.

---

## 📌 Project Overview

Smart To-Do List Pro allows users to:
- Add new tasks
- View all tasks
- Mark tasks as completed
- Delete tasks
- Filter tasks (Pending / Completed)
- Save tasks permanently using a JSON file

---

## 🚀 Features

### Core Features
- Add tasks
- View task list
- Mark tasks as completed
- Delete tasks

### Advanced Features
- Persistent storage using JSON file
- Task filtering (Pending / Completed)

### Creative Feature
- Status tracking system
- Simple and user-friendly menu interface
